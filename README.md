# 🚚 Round Rock – Trash Hauling Landing Page

Round Rock is a modern React-based project built with Vite, styled with TailwindCSS and Flowbite, and featuring reactive forms, data validation and alerts. This project focuses on clean architecture, modular components, and a responsive UI.

## 🚀 Features

- React + Vite: Fast and modern frontend development.
- TailwindCSS + Flowbite: Beautiful responsive UI with prebuilt components.
- Responsive layout (mobile, tablet, desktop)
- Interactive Hero section with clipped images
- Reusable Components: Section titles, service cards, testimonials
- Form validation with React Hook Form + Zod
- Email form integration with Axios and SweetAlert2
- Animated interactions using Tailwind transitions and Flowbite UI
- Alerts & Notifications: User-friendly alerts using sweetalert2.
- Icons: Clean, customizable icons via lucide-react.

---

## 🛠️ Tech Stack

- Frontend Framework: React 19
- Build Tool: Vite 7
- Styling: TailwindCSS  + Flowbite  + Flowbite React
- Form Validation: React Hook Form  + Zod 
- HTTP Requests: Axios 
- Icons: Lucide 
- Alerts: SweetAlert2
